Bti - Конфигурация импорта данных из БТИ
 - MappingConfiguration - настройка соответствия справочников БТИ и ОПС
 - BtiObjectsImportConditions - условия переноса данных из БТИ в ОПС + общие настройки наименований таблиц в источнике данных
Common - настройка меню системы
Mfc - настройки импорта данных из МФЦ
Param - различные настройки
 - Core.DataBase.Connections - настройка подключений к БД
 - Core.Diagnostics - настройка диагностики
 - Core.DocNumeration - не используется
 - Core.FileStorages - настройка различных файловых хранилищ
 - Core.Numeration - настройка нумераторов системы
 - FmReportTypes - настройка параметров отчетов
 - InsurObjectFlagInsur - настройка условий для признака Подлежит страхованию
 - InsurObjectMapBuilding - настройка соответствия полей МКД с источниками БТИ и ЕГРН
 - InsurObjectMapFlat - настройка соответсвия полей квартиры с источниками БТИ и ЕГРН
 Registers - настройка представлений реестров
 Reports - шаблоны отчетов FastReport
