
Для запуска генерации ОРМ нужно выбрать ключ в app.config (сейчас доступны ключи: General, Analogs, Modeling).


Для формирования ОРМ в отдельном проекте нужно:
1. Добавить ключ в app.config (<add key="Mode" value="Modeling"/>)
2. Добавить объект с настройками в appsettings.json
3. Если проект создается впервые, то запустить утилиту с ключем General (чтобы объекты удалились из старого файла)
